# Wattar Academy - User Stories

## Manager (Academy Owner/Director)

### Student Management
- As a manager, I want to see all students in the academy so I can monitor enrollment
- As a manager, I want to add new students with their details (name, ID, level, start date) so I can track enrollment
- As a manager, I want to edit student information so I can keep records updated
- As a manager, I want to see which trainer is assigned to each student

### Attendance & Performance
- As a manager, I want to see academy-wide attendance statistics so I can monitor overall performance
- As a manager, I want to see attendance by trainer so I can evaluate trainer effectiveness
- As a manager, I want to see which students have poor attendance so I can take action

### Financial Overview
- As a manager, I want to see total monthly revenue so I can track business performance
- As a manager, I want to see outstanding payments so I can follow up with students
- As a manager, I want to track trainer salaries and payments

### Reporting
- As a manager, I want to export reports to Excel so I can share with stakeholders
- As a manager, I want to see monthly/quarterly summaries for business planning

---

## Reception Staff

### Student Management
- As reception staff, I want to register new students quickly so I can handle enrollment efficiently
- As reception staff, I want to search for students by name or ID so I can find them quickly
- As reception staff, I want to update student contact information when they inform me

### Payment Management
- As reception staff, I want to record student payments so I can track who has paid
- As reception staff, I want to see which students have outstanding balances so I can remind them
- As reception staff, I want to print payment receipts for students

### Daily Operations
- As reception staff, I want to see today's class schedule so I can inform students
- As reception staff, I want to mark attendance if a trainer is absent so classes aren't missed

---

## Trainers

### Class Management
- As a trainer, I want to see my assigned students so I know who I'm teaching
- As a trainer, I want to see my class schedule so I know when to teach

### Attendance
- As a trainer, I want to mark attendance for my classes quickly so I don't waste class time
- As a trainer, I want to mark students as present/absent/late so I can track participation
- As a trainer, I want to see attendance history for my students so I can identify patterns

### Student Progress
- As a trainer, I want to add progress notes for students so I can track their development
- As a trainer, I want to recommend level changes when students are ready to advance
- As a trainer, I want to see how long each student has been in their current level

### Personal Dashboard
- As a trainer, I want to see my teaching statistics (total students, attendance rates) so I can track my performance
- As a trainer, I want to access the system from my phone so I can mark attendance anywhere

---

## Parents (Future Phase)

### Student Monitoring
- As a parent, I want to see my child's attendance record so I can monitor their participation
- As a parent, I want to see my child's progress notes so I know how they're developing
- As a parent, I want to see payment history and outstanding balances
- As a parent, I want to receive notifications about missed classes

---

## System Administrator (Manager)

### User Management
- As an admin, I want to create accounts for trainers and reception staff
- As an admin, I want to assign students to trainers
- As an admin, I want to backup data regularly so we don't lose information

---

## Phase 1 Priority User Stories

### High Priority (Must Have)
1. **Manager:** Student management, basic reporting
2. **Reception:** Student registration, payment recording  
3. **Trainers:** Class attendance marking, student list viewing

### Medium Priority (Should Have)
1. **Manager:** Attendance statistics by trainer
2. **Reception:** Outstanding balance tracking
3. **Trainers:** Progress note adding

### Low Priority (Could Have)
1. **Manager:** Advanced financial reporting
2. **Reception:** Class schedule viewing
3. **Trainers:** Personal performance statistics
